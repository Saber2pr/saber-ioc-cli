# saber-ioc-cli

> a cli for saber-ioc dev

```bash
# from npm
npm install saber-ioc-cli -g

# from github
git clone https://github.com/Saber2pr/saber-ioc-cli.git
```

# Usage

1. sioc `<Module Name>`

For example:

```bash
sioc Hello
```

---

## start

```bash
npm install
```

```bash
npm start

npm run test

```
